🔍 API Capture & Analyzer

自动捕获和标注浏览器页面中的所有网络接口

🎯 核心功能

实时捕获页面中的所有网络请求，包括XHR、Fetch等，并智能识别接口类型。

🏷️ 智能标注

自动识别RESTful API、数据接口、文件接口等多种类型，并提供详细分析。

📊 实时统计

显示总请求数、独立域名数、API接口数等统计信息，支持实时更新。

📤 数据导出

将捕获的网络请求数据导出为JSON格式，便于分析和存档。

📋 测试说明

如何测试

  1. 确保已安装 API Capture 扩展
  2. 打开浏览器开发者工具（F12）
  3. 切换到 Console 标签页
  4. 点击上方的测试按钮
  5. 观察扩展是否捕获了请求

预期结果

  • 扩展图标应显示捕获的请求数
  • 点击扩展图标应显示请求列表
  • 每个请求应有正确的类型标注
  • 请求详情应包含完整信息

⚠️ 常见问题

Q: 请求没有被捕获？

A: 确保扩展已启用，且在扩展管理页面中设置了正确的权限。

Q: 类型识别不准确？

A: 扩展使用URL模式匹配，某些非标准API可能需要自定义规则。

Q: 数据没有实时更新？

A: 尝试刷新扩展弹出窗口，或重新加载测试页面。
